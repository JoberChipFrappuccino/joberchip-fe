## 실행 방법

### Developmenet

```bash
# 번들링
npm run build:dev

# 서버 실행
npm run dev
```

### Production

```bash
# 번들링
npm run build:prod

# 서버 실행
npm run start
# 또는
npm run start:pm2
```
